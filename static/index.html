SocketBox!

SocketBox is an easy to use web service with a simple API which offers push notifications to your web apps from your server . It simplifies the work of a developer to push out real time information to clients . It is highly scalable and flexible and it is supported to work on a wide range of environments .

Simple Yet Flexible!

Almost all applications today are highly dynamic and realtime . It is highly important to send notifications in realtime from your server to all your web clients without the need to poll. This makes your web application highly responsive and dynamic . SocketBox provides just the righ tool for you to achieve this kind of realtime information broadcasting .

Ready To Use!

We believe that developers should not waste their time setting up basic elements like socket communications for real time information sharing , instead developers should spend their time and resources on building the cool stuff . Thus SocketBox came to life and it provides a simple and easy to use API to quickly get started on giving out push notifications .
